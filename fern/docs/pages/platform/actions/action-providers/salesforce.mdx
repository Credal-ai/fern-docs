# Salesforce

The Salesforce connector allows you to manage records, create cases, and generate reports efficiently.

## Supported Actions

### Update a Record in Salesforce

**Description:** Update an existing record in Salesforce to keep information current and accurate.

### Create a Case or Support Ticket in Salesforce

**Description:** Create a new case or support ticket in Salesforce to track customer issues or requests.

### Generate a Sales Report from Salesforce

**Description:** Generate a sales report from Salesforce to analyze sales data and performance.

### Retrieve a Record from Salesforce

**Description:** Retrieve a specific record from Salesforce for review or processing.

### Retrieve Records by SOQL Query

**Description:** Retrieve records from Salesforce using SOQL (Salesforce Object Query Language) for advanced querying capabilities.
