# Slack

The Slack connector allows you to interact with Slack by listing conversations and sending messages to channels.

## Supported Actions

### List All Conversations in Slack Workspace

**Description:** Retrieve a list of all conversations within the Slack workspace.

### Send a Message to a Slack Channel

**Description:** Send a message to a specified Slack channel.  
**Recommendation:** Input the name of the channel without a hashtag.
