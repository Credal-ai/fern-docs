# Composio Integration (Beta)

[Composio](https://www.composio.dev/) is a platform that provides a unified interface for connecting applications with various third-party services and tools. When you use Composio actions through Credal, you gain access to a vast library of pre-built actions and tools that can be seamlessly incorporated into your AI agents.

Composio is currently unavailable for use, but will be released in beta mode at the end of March or beginning of April 2026.

You will be able to use Composio actions through Credal in exactly the way that you currently use other actions. The first Composio toolkits we will support are:

* GitHub
* Stack Overflow
* Google Docs
* Fireflies
* Ashby
* Zoom

When we launch this feature, you will be able to ask your Credal admin to enable actions from any of those providers. If there are actions available in any of the available [toolkits](https://composio.dev/toolkits/github) above that you'd like to use but are not available to your admin, reach out to support@credal.ai so we can turn those on in our product. Similarly, if any other [toolkits](https://composio.dev/toolkits?utm_campaign=toolkits) interest you, please reach out to support@credal.ai and we can easily integrate them into our product.
