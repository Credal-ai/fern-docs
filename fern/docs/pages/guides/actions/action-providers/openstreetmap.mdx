# OpenStreetMap
The OpenStreetMap connector allows you to obtain geographical coordinates for specific locations.

## Authentication Setup
**Setup:** This is set up via Credal, no setup needed.

## Supported Actions

### Get the Latitude and Longitude of a Location
**Description:** Retrieve the latitude and longitude of a specified location to use in mapping or location-based services.  
**Recommendation:** Provide a clear and specific location name or address to ensure accurate coordinate retrieval.
