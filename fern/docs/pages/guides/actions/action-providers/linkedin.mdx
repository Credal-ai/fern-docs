# LinkedIn
The LinkedIn connector allows you to interact with LinkedIn by creating shareable post links efficiently.

## Authentication Setup
**Requirement:** Organization-wide credential  
**Setup:** This is set up via Credal, no setup needed. It is suggested to be signed into LinkedIn for minimal disruption.

## Supported Actions

### Create a Share LinkedIn Post Link
**Description:** Create a shareable link for a LinkedIn post to easily distribute content across platforms.
