# Finnhub

The Finnhub connector allows you to interact with financial data by looking up stock symbols and retrieving company financials efficiently.

## Authentication Setup
- **Requirement:** Organization-wide credential
- **Setup:** This uses a Credal API key, no setup needed

## Supported Actions

### Look Up a Stock Symbol by Name
- **Description:** Retrieve the stock symbol for a company by its name
- **Recommendation:** The AI is pretty good with knowing stock symbols, but recommend putting in a select few options for greater control

### Get Company Basic Financials
- **Description:** Retrieve basic financial information for a company, such as margin, P/E ratio, 52-week high/low, etc.
