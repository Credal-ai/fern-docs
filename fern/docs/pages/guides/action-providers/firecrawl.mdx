# Firecrawl
The Firecrawl connector allows you to interact with web pages by scraping content efficiently.

## Authentication Setup
**Requirement:** Organization-wide credential  
**Setup:** This is set up via Credal, no setup needed

## Supported Actions

### Scrape Tweet Data with Nitter and Firecrawl
**Description:** Scrape tweet data using Nitter and Firecrawl to gather information from Twitter without direct API access.  
**Recommendation:** Ensure you have the correct URL and parameters for the tweets you wish to scrape.

### Scrape a URL and Get Website Content Using Firecrawl
**Description:** Scrape a URL to retrieve website content using Firecrawl.  
**Recommendation:** Verify the URL is correct and accessible to ensure successful content retrieval.
