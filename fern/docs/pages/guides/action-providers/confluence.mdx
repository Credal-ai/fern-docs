# Confluence

The Confluence connector allows you to interact with Confluence pages by updating content or fetching existing content efficiently.

## Authentication Setup
- **Requirement:** Organization-wide credential
- **Setup:** Contact a Credal admin to set up a Credal Service Account API connection

## Supported Actions

### Update a Confluence Page
- **Description:** Update a Confluence page with new content specified
- **Recommendation:** If you know the page won’t change, we recommend setting the ID here.

### Fetch Content from a Confluence Page
- **Description:** Fetch content from a Confluence page
- **Recommendation:** If you know the page won’t change, we recommend setting the ID here.
