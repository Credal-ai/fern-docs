# NWS (National Weather Service)
The NWS connector allows you to retrieve weather forecasts for specific locations using latitude and longitude.

## Authentication Setup
**Setup:** This is set up via Credal, no setup needed.

## Supported Actions

### Get the Weather Forecast for a Location Using Latitude and Longitude
**Description:** Retrieve the weather forecast for a specific location by providing its latitude and longitude coordinates.  
**Recommendation:** Ensure the coordinates are accurate to receive the correct weather information for the desired location.
