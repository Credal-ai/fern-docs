# Google Maps
The Google Maps connector allows you to interact with Google Maps by validating addresses and searching for nearby places efficiently.

## Authentication Setup
**Requirement:** Organization-wide credential  
**Setup:** This is set up via Credal, no setup needed

## Supported Actions

### Validate a Google Maps Address
**Description:** Validate an address using Google Maps to ensure it is accurate and exists within the map database.  
**Recommendation:** Double-check the address format and details for accurate validation.

### Search for Nearby Places Using Google Maps
**Description:** Search for places near a specified location using Google Maps to find points of interest or services.  
**Recommendation:** Specify the location and type of places you are interested in to refine your search results.
