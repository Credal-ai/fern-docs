# GSuite
The GSuite connector allows you to interact with Google applications like Google Docs and Google Calendar using OAuth for authentication.

## Authentication Setup
**Requirement:** OAuth credentials  
**Setup:** Instructions coming within the week!

## Supported Actions

### Create a New Google Doc Using OAuth
**Description:** Create a new Google Doc using OAuth to authenticate and authorize access to your Google Drive.  
**Recommendation:** Ensure you have the necessary permissions and OAuth setup to create and manage documents.

### Schedule a Meeting on Google Calendar with OAuth
**Description:** Schedule a meeting on Google Calendar using OAuth to authenticate and manage your calendar events.  
**Recommendation:** Confirm the meeting details and participant availability before scheduling to avoid conflicts.
