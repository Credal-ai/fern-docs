# Looker
The Looker connector allows you to manage and interact with Looker users efficiently.

## Authentication Setup
**Setup:** Today, this requires an org-wide credential. Reach out to a Credal admin to set up a connection.

## Supported Actions

### Search for a Looker User by Email and Enable Them if Disabled
**Description:** Search for a Looker user by their email address and enable their account if it is currently disabled.
