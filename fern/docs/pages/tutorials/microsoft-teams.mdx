---
title: Microsoft Teams Setup
slug: microsoft-teams
---

1. Install the application on Microsoft Teams.
2. Set up your Credal account [here](https://app.credal.ai/). (Make sure the domain for your email in Microsoft Teams matches the domain you use when signing up for Credal, i.e. use&#x72;**@example.com**)
3. [Create a new agent](https://docs.credal.ai/user-guide/getting-started/quickstart) in Credal.
4. (Optional) Set your default agent [here](https://app.credal.ai/admin/customization).
5. Message the app in Microsoft Teams and your default agent will respond to your queries.
