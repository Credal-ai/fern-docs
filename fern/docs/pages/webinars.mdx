## Getting Started

Not sure which training is right for you?

- **Start with Beginner Training** if you're new to Credal or have never built an agent
- **Jump to Advanced Training** if you've built agents and want to learn about actions and deployment

Both sessions include hands-on demonstrations and practical examples you can follow along with.
