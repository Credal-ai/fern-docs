## Getting Started

Not sure which webinar is right for you? Here's a quick guide:

- **Choose the Beginner Webinar** if you're new to Credal or have never built an agent before
- **Choose the Advanced Webinar** if you've already experimented with agent building and want to learn about actions, multi-agent systems, and deployment

Both webinars are hands-on and include live demonstrations, Q&A sessions, and practical exercises you can follow along with.
