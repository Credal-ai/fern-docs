# Audit Log / Data Exports

Credal provides comprehensive audit log and data export functionality, enabling you to export audit logs and platform data to your own data warehouse or analytics platform for compliance, security monitoring, and analysis purposes. *For user provisioning and access control, see&#x20;*[*Single Sign-On (SSO)*](/user-guide/platform/administration/single-sign-on-sso)*.*

This functionality is built on Fivetran's Hybrid Deployment offering, ensuring all data stays resident within accounts managed by Credal or your own organization, maintaining complete data sovereignty.

In addition to audit log exports, other types of data exports are available e.g. agent metadata. Contact your Credal representative for more information about additional export options.

## Available Destinations

- Azure SQL Database, Azure SQL Managed Instance
- Azure Synapse
- AWS MSK
- Databricks
- Google BigQuery
- MySQL (native server)
- MySQL (Cloud-Hosted: AWS RDS, AWS RDS Aurora)
- Oracle (Cloud-Hosted: AWS RDS)
- PostgreSQL (native server)
- PostgreSQL (Cloud-Hosted: AWS RDS, AWS RDS Aurora, Google Cloud)
- Redshift
- Snowflake
- MariaDB (native server)
- MariaDB (Cloud-Hosted: AWS RDS)
- Microsoft SQL Server (native server)
- Microsoft SQL Server (Cloud-Hosted: Azure)

Additional destination options may be available.
